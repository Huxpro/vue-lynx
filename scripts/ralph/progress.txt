# Vue-Lynx Chinese Translation Progress

## Project Context
- Rspress i18n: https://rspress.rs/guide/basic/i18n
- Source: website/docs/
- Target: website/docs/zh/

## Translation Guidelines
- Keep code blocks, URLs, file paths in English
- component → 组件
- props → props (keep English)
- state → 状态
- lifecycle → 生命周期
- template → 模板
- directive → 指令
- slot → 插槽
- main thread → 主线程

## Learnings
- Dev server runs on port 3000, not 5173
- Rspress v2.0.5 language switcher is a hover dropdown in the navbar
- Default language docs stay at root (docs/), translations go in subdirectory (docs/zh/)
- locales config in top-level defineConfig activates language switcher automatically
- themeConfig.locales allows per-locale nav overrides
- Path-based sidebar keys (/zh/guide/) work for locale-specific sidebar
