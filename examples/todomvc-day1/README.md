# TodoMVC (Day 1 Tribute)

This is the Day 1 tribute version of [TodoMVC](https://todomvc.com) built with Vue 3 × Lynx using basic CSS Selector styling, before we iteratively improved the CSS to match the pixel-perfect `todomvc-codex` reference.

## Features Exercised

- `<script setup>` with `defineProps` / `defineEmits`
- `ref`, `computed` for state management
- `v-for` list rendering with `:key`
- `v-if` / `v-else` conditional rendering
- Dynamic `:class` binding
- `@tap`, `@longpress`, `@confirm`, `@blur` events
- CSS Selectors (`enableCSSSelector: true`)
- Multi-component composition (TodoApp, TodoHeader, TodoItem, TodoFooter)
