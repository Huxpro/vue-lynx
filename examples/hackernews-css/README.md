# HackerNews Slopfork (Original CSS)

A faithful Vue Lynx port of the classic HackerNews example, retaining the original SCSS styles from the Vue 3 reference implementation.

## Quick Start

```bash
pnpm install
pnpm run dev
```

## Features

- **Original Styles**: Proves Vue Lynx's robust support for traditional web CSS (`.class` selectors, nesting, CSS variables)
- **Vue Router**: Client-side routing with `createMemoryHistory()`
- **Pinia**: State management for user and feed data
- **Vue Query**: Server state caching and prefetching
- **Transitions**: Slide animations between pages
