# 7 GUIs

The [7 GUIs](https://eugenkiss.github.io/7guis/) benchmark defines seven tasks that represent typical challenges in GUI programming — from a simple counter to a full spreadsheet. Here we implement all seven using Vue Lynx, forked from the [official Vue.js examples](https://vuejs.org/examples/#7-guis-counter).
