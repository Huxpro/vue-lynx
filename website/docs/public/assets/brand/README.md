# Vue Lynx brand marks

The Vue Lynx logo follows the Lynx family system established by ReactLynx:
the framework's own iconic mark carries the shared **Lynx lightning bolt** as
its energy core. Here, Vue's two-tone chevron seats the *exact* Lynx bolt path
(the same one ReactLynx uses at its nucleus) in the notch.

| File | Use |
| --- | --- |
| `vue-lynx-logo.svg` | Primary full-color mark. Works on light, dark, and Vue-green grounds (the bolt is white on the slate inner chevron). |
| `vue-lynx-logo-mono.svg` | Single-ink version — the bolt is knocked out as transparent negative space. Fills with `currentColor`, so it inherits any brand tint. Set `color` (when inlined) or `fill` to color it; on dark grounds set it to a light value. |

## Colors

- Vue green — `#42B883`
- Vue slate — `#35495E`
- Bolt — `#FFFFFF` (primary) / knockout (mono)

## Clear space & minimum size

Keep clear space of at least the width of the bolt on all sides. The mark stays
legible down to 32 px; below that, prefer a solid-tile / favicon treatment.
