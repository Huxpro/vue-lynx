# 7GUIs 基准测试

为了验证 Vue Lynx 在真实 GUI 场景中与 Vue 行为一致，我们从 [Vue.js 官方文档](https://vuejs.org/examples/#7-guis-counter) Fork 了 [7 GUIs](https://eugenkiss.github.io/7guis/) 示例 —— 七个经典任务，涵盖了 GUI 编程中的典型挑战，从简单的计数器到完整的电子表格。

## 示例

一个简单的按钮，用于递增计数器。测试基本的状态管理和事件处理。
