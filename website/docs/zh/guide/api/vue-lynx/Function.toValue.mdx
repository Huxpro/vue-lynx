# Function: toValue()

```ts
function toValue<T>(source): T
```

Normalizes values / refs / getters to values.
This is similar to [unref](Function.unref.mdx), except that it also normalizes getters.
If the argument is a getter, it will be invoked and its return value will
be returned.

## Returns

`T`

## Example

```js
toValue(1) // 1
toValue(ref(1)) // 1
toValue(() => 1) // 1
```
